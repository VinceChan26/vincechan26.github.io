改寫 commit

使用時機：

  • 增加最近提交時漏掉的檔案
  • 修改最近的提交的內容或註解

如何寫好 Commit Message

  • 用新的空行將標題和主體兩者隔開。
  • 標題首字母大寫
  • 不要用句號結束標題
  • 標題中使用祈使句
  • 主體部分去解釋「是什麼」和「為什麼」而不是怎麼做
